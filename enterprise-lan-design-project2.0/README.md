# Enterprise LAN Network Simulation (Improved Version)

This project represents an improved network architecture based on the previous project **enterprise-lan-design-project**.

The goal of this simulation was to extend the original enterprise network by adding better segmentation, additional services, and a more realistic infrastructure model.

The network was designed and simulated using Cisco Packet Tracer.

## Key Improvements

- Network segmentation using multiple VLANs
- Dedicated networks for different departments
- Improved infrastructure for internal services
- Backup server integration
- NAS storage integration
- Wi-Fi infrastructure for management areas
- Department isolation for security and traffic control
- Improved network topology and device organization

## Network Departments

The network includes several simulated company departments:

- Administration
- Executive Office
- Helpdesk / IT Support
- Sales
- Maintenance
- Accounting and Coordination
- Internal Servers
- Web Services

## VLAN Structure

The network is segmented using VLANs to improve performance and security.

Example VLAN structure:

- VLAN 1 – Helpdesk
- VLAN 2 – Administration
- VLAN 3 – Accounting and Coordination
- VLAN 5 – Sales
- VLAN 6 – Maintenance
- VLAN 7 – Internal Servers
- VLAN 8 – Web Server Network
- VLAN 99 – IT Management

- Cisco Packet Tracer

- Network topology design
- Enterprise network simulation

# Simulação de Rede Empresarial LAN (Versão Melhorada)

Este projeto representa uma versão melhorada da arquitetura de rede desenvolvida no projeto **enterprise-lan-design-project**.

O objetivo desta simulação foi expandir a rede empresarial original, adicionando uma melhor segmentação da rede, novos serviços e uma infraestrutura mais próxima de um ambiente empresarial real.

A rede foi desenhada e simulada utilizando o Cisco Packet Tracer.

## Melhorias Implementadas

- Segmentação da rede através de VLANs
- Redes dedicadas para diferentes departamentos
- Infraestrutura melhorada para serviços internos
- Integração de servidor de backup
- Integração de armazenamento NAS
- Infraestrutura Wi-Fi para áreas administrativas
- Isolamento de departamentos para segurança e controlo de tráfego
- Organização melhorada da topologia de rede

## Departamentos Simulados

A rede inclui vários departamentos de uma empresa:

- Administração
- Gabinete do CEO
- Helpdesk / Suporte Técnico
- Vendas
- Manutenção
- Coordenação e Contabilidade
- Servidores Internos
- Serviços Web

## Estrutura de VLANs

A rede foi segmentada através de VLANs para melhorar desempenho e segurança.

Exemplo de estrutura:

- VLAN 1 – Helpdesk
- VLAN 2 – Administração
- VLAN 3 – Coordenação e Contabilidade
- VLAN 5 – Vendas
- VLAN 6 – Manutenção
- VLAN 7 – Servidores Internos
- VLAN 8 – Servidor Web
- VLAN 99 – Gestão de TI

## Tecnologias Utilizadas

- Cisco Packet Tracer

# Enterprise IT Lab

## 🇬🇧 Overview

This repository showcases hands-on IT infrastructure and networking projects, focused on real-world enterprise scenarios.

The projects go beyond basic academic requirements, including advanced configurations, troubleshooting, and infrastructure design.

### 🔧 Key Skills Demonstrated

* Enterprise network design (LAN/WAN)
* VLAN segmentation and network isolation
* pfSense firewall configuration with IDS/IPS
* VPN implementation (site-to-site and remote access)
* Dynamic routing (RIP)
* Active Directory deployment and management
* Virtualization and multi-VM environments
* Network troubleshooting and problem-solving

### 🧱 Featured Projects

* **Enterprise LAN Design Project**
  Design and implementation of a scalable enterprise network with redundancy, VLANs, and centralized services.

* **Final Network Infrastructure Project (209 pages)**
  Advanced infrastructure including dual ISP failover, VPNs, routing, and security mechanisms.

* **Linux Infrastructure Lab (ongoing)**
  Deployment of Linux-based services such as DHCP, web server, and file sharing in a virtualized environment.

### 📊 Project Assets

Each project may include:

* Technical documentation (PDF reports)
* Network diagrams (Visio)
* Packet Tracer simulations
* Screenshots and configurations

## 🇵🇹 Visão Geral

Este repositório apresenta projetos práticos de infraestrutura IT e redes, focados em cenários reais de ambiente empresarial.

Os projetos vão além dos requisitos académicos básicos, incluindo configurações avançadas, troubleshooting e desenho de arquitetura de rede.

### 🔧 Competências Demonstradas

* Desenho de redes empresariais (LAN/WAN)
* Segmentação por VLAN e isolamento de rede
* Configuração de firewall pfSense com IDS/IPS
* Implementação de VPN (site-to-site e acesso remoto)
* Routing dinâmico (RIP)
* Implementação e gestão de Active Directory
* Virtualização e ambientes com múltiplas VMs
* Troubleshooting de rede

### 🧱 Projetos em Destaque

* **Enterprise LAN Design Project**
  Planeamento e implementação de uma rede empresarial escalável com redundância, VLANs e serviços centralizados.

* **Projeto Final de Infraestrutura de Rede (209 páginas)**
  Infraestrutura avançada com dual ISP, VPNs, routing e mecanismos de segurança.

* **Laboratório de Infraestrutura Linux (em desenvolvimento)**
  Implementação de serviços Linux como DHCP, web server e partilha de ficheiros em ambiente virtualizado.

### 📊 Conteúdo dos Projetos

Cada projeto pode incluir:

* Documentação técnica (relatórios em PDF)
* Diagramas de rede (Visio)
* Simulações no Packet Tracer
* Screenshots e configurações
